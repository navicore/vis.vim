vis.vim
=======

vis.vim plugin for vundle installation

see doc dir for author and license info

update 3/2025
-----------
I still use this in my neovim setup.

Lazy config:
========

```lua
{
  'navicore/vis.vim',
}
```

I suggest using `vis` with `vissort`:

```lua
{
  'navicore/vis.vim',
  'navicore/vissort.vim',
}
```
